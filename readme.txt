This asset contains source codes for pre-processing extracted Sentinel-2 satellite data for using in Geo spatial Prithvi Crop Classification problem. Data for three time steps need to be downloaded and used.

This folder contains two sub folders:
1. Data_Japan        - Downloaded from Sentinel-2 three months data for the year of interest
2. src 	             - consists of the source file Preprocess_Sentinel2ToPrithviCrop.py. It also consists of one tif file, which is required as reference for pre-processing.
3. requirements.txt. - consists of the list of packages
